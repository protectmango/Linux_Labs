# Lab 01 – TCP Echo Server

## 🧭 Objective
- Learn **TCP socket programming** in C.  
- Implement a **server** that echoes client messages.  
- Understand `socket()`, `bind()`, `listen()`, `accept()`, `read()`, `write()`.  

## ⚙️ Steps
1. Compile server and client:
```bash
make
make run-server
make run-client
```
